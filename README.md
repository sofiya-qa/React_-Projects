# Admin Dashboard (React + Fluent UI)

A simple **Admin Dashboard** built with **React** and **Fluent UI** that allows an admin to:
- View a list of users in a table.
- Search users by name.
- Add new users.
- Edit existing users.
- Delete users with confirmation.
